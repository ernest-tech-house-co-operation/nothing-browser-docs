# TECH HOUSE Tab

The update and changelog panel for Nothing Browser.

## What's on This Tab

- **Current version** indicator
- **Update status** — live check result from GitHub Releases API
- **Download / Install controls** for in-app updates
- **Changelog** — shows the full changelog from the latest release

## Quick Access Buttons

- **OPEN DEVTOOLS / SCRAPPER** — switches to the DEVTOOLS tab
- **OPEN BROWSER** — switches to the BROWSER tab

## Update Bell

When a new version is available, the notification bell (🔔) in the top-right of the tab bar turns amber. The tab label updates to:

```
🔔 TECH HOUSE [v0.1.4 ready]
```

See [Auto-Update](/guide/auto-update) for full details on the update flow.

## Stats Panel

Shows:
- Current version
- Engine (Chromium/Qt6)
- Platform (Linux / macOS / Windows)
