# PLUGINS Tab

The plugin manager for Nothing Browser.

## INSTALLED Sub-Tab

Shows all installed plugins with:
- Name, author, version
- Enable / Disable toggle
- Uninstall button
- Description and how-to-use text

Click **+ FROM FOLDER** to install a plugin from a local directory.

## COMMUNITY Sub-Tab

Loads the community plugin registry from:
`github.com/ernest-tech-house-co-operation/nothing-browser-plugins`

Click **↺ REFRESH** to fetch the list. Select any plugin to see its details, then click **↓ INSTALL** to install it directly.

Already-installed plugins are shown with an `[INSTALLED]` label.
