# Privacy Policy

**Last Updated:** April 24, 2026

---

## Overview

Nothing Ecosystem respects your privacy. **The Nothing Browser and Nothing Private Browser binaries send zero data out of your machine.**

This Privacy Policy explains what data we collect, what we don't collect, and what leaves your machine.

**The short version:** Nothing leaves your machine except the websites you visit. We don't phone home. We don't send telemetry. We don't track you.

---

## Nothing Browser (Scraper Browser)

### Data That Never Leaves Your Machine

| Data Type | Does it leave? |
|-----------|----------------|
| Browsing history | ❌ No |
| Cookies | ❌ No |
| Saved passwords | ❌ No |
| Bookmarked sites | ❌ No |
| Session data | ❌ No |
| Network captures | ❌ No |
| Fingerprint configuration | ❌ No |
| Scraped data | ❌ No |
| API responses | ❌ No |
| Request logs | ❌ No |

### What Does Leave Your Machine

**Only the websites you visit.**

When you navigate to a website using Nothing Browser, your browser makes network requests to that website. This is the intended purpose of the software.

```
Your Machine ──────► Website You Requested
              │
              └─── No other destination. Ever.
```

**We do not intercept, log, or store these requests.** The request goes directly from your browser to the website.

### What We Don't Do

| Activity | Status |
|----------|--------|
| Send telemetry | ❌ Never |
| Phone home | ❌ Never |
| Check for updates automatically | ❌ Never |
| Send crash reports | ❌ Never |
| Track your browsing | ❌ Never |
| Log your IP address | ❌ Never |
| Embed analytics | ❌ Never |
| Use tracking pixels | ❌ Never |

**Zero data leaves your machine except the websites YOU choose to visit.**

The Private Browser is designed for maximum privacy:

- ✅ No persistent storage
- ✅ No session persistence
- ✅ No telemetry
- ✅ No update checks
- ✅ No phone home
- ✅ Zero external calls except your explicit navigation

## Data Storage Locations

### Nothing Browser (Scraper Browser)

| Data | Location |
|------|----------|
| Profiles | `~/.local/share/nothing-browser/` (Linux) |
| Session data | `~/Library/Application Support/nothing-browser/` (macOS) |
| Cookies | Local profile directory |
| Exported sessions | Your chosen export path |

### Nothing Private Browser

| Data | Location |
|------|----------|
